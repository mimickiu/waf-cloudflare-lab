Cloudflare WAF Laboratory

Práctica de protección de aplicaciones web

Objetivo

Este laboratorio demuestra el funcionamiento de un Web Application Firewall (WAF) utilizando Cloudflare.

¿Qué es un WAF?

Un WAF inspecciona solicitudes HTTP y HTTPS para detectar y bloquear ataques contra aplicaciones web.

Protecciones Analizadas

  • SQL Injection (SQLi)
  • Cross-Site Scripting (XSS)
  • Rate Limiting
  • Bot Protection
  • Managed Rules
  • Custom Rules

Estado

Laboratorio activo.

Servidor protegido por Cloudflare.

Aviso

Este entorno tiene fines educativos. Toda actividad puede ser registrada para fines de monitoreo y auditoría.
